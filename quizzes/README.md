# Review Questions

Quick self-test before moving on.

## How to Use

1. Open `01_python_review.py` in your editor
2. Run `python run_review.py`
3. Fill in the `FILL_ME_IN` blanks to fix failing tests
4. Repeat until done

## Commands

```bash
python run_review.py          # Run tests
python run_review.py --hint   # Get help
python run_review.py --all    # See all results
```

## Topics

- **Q01-10**: Python basics (lists, dicts, slicing, loops)
- **Q11-20**: NumPy (arrays, shapes, operations)
- **Q21-25**: Matplotlib (plotting, images)
