# DMLAP Koans

Test your Python knowledge by filling in the blanks.

## How to Use

1. Open `01_python_koans.py` in your editor
2. Run the tests: `python run_koans.py`
3. Find the first failing test and fix it by replacing `__` with the correct code
4. Run again. Repeat until all tests pass.

## Commands

```bash
python run_koans.py          # Run tests, show first failure
python run_koans.py --hint   # Get a hint for current failure
python run_koans.py --all    # Show all pass/fail results
```

## Topics Covered

- **Koans 01-10**: Python basics (lists, dicts, slicing, loops)
- **Koans 11-20**: NumPy (arrays, shapes, operations)
- **Koans 21-25**: Matplotlib (plotting, images)
